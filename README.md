# 产品新人术语地图

一句话介绍：一个面向产品新人、AI 产品运营实习准备者和产品助理候选人的产品术语学习网页。

## 项目说明

这个仓库包含一个单文件静态网页 `index.html`，用于整理常见产品经理术语、行业表达和 AI 产品相关概念。网页支持关键词搜索、难度筛选、类别筛选和术语卡片翻转，适合作为作品集辅助项目或新人学习工具。

## 声明

本项目为个人学习与作品集展示用途。术语内容基于公开资料与个人理解重新整理，旨在帮助产品新人快速建立基础概念，不代表权威定义。

## 内容范围

- 用户与需求类
- 产品体验与功能类
- 产品文档与流程类
- 数据与运营类
- 商业与增长类
- AI 产品专项类

## 使用方式

直接打开 `index.html` 即可浏览网页，不需要安装依赖，也不需要后端服务。
